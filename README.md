Project Overview
This platform serves as a digital hub for coordinating taxi services. It allows passengers to view available routes and stations while providing administrators with a powerful dashboard to manage the entire logistics process, including trips, drivers, and stations.

✨ Key Features
Dynamic Routing: Real-time display of available trips based on origin and destination.

Booking Management: Efficient handling of passenger reservations and seat availability.

Administrative Dashboard: A comprehensive Django-powered admin panel for full CRUD operations on trips and stations.

Structured Database: A well-designed relational schema for storing user, driver, and trip data.

Cloud Deployment: Fully deployed and operational on PythonAnywhere.

🛠 Tech Stack
Backend: Python 3.10 & Django Framework.

Frontend: HTML5, CSS3, and JavaScript.

Database: Relational database (SQLite/PostgreSQL).

Deployment: Hosted via PythonAnywhere.
